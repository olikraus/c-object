License: CC BY-SA 4.0, Attribution-ShareAlike 4.0 International, https://creativecommons.org/licenses/by-sa/4.0/ 


## Features

 * Generic objects implemented in C
 * Support for vector and map data structures
 * JSON read and write
 * GZIP support

## Useful Tools

 * `csv2json`: A tool to convert csv files to JSON format
 * `json_compare`: Compre two different JSON files and output path to the first difference
 * `json_format`: A simple beautifier tool for JSON
